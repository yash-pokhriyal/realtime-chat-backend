# Realtime Chat Backend

A production-grade Slack/Discord inspired backend built with Go.

## Features

- User Registration
- User Login
- JWT Authentication
- Protected Routes
- Password Hashing (bcrypt)
- PostgreSQL Database
- GORM ORM
- Repository Pattern
- Real-Time Chat using WebSockets
- Broadcast Messaging

## Tech Stack

- Go
- Gin
- PostgreSQL
- GORM
- Gorilla WebSocket
- JWT
- bcrypt
